Music Explorer

Music Explorer is an application that allows users to procedurally explore related artists of a searched-upon artist. An app built with React, the D3 visuals library, and a few API's to handle artist recommendations and rendering of song previews, this project was the term project for a previous front-end web development course.

Terminal

The Terminal application was developed in tandem with my group as the term project for the agile course I took at PSU. Written in Python and using Pysftp, this application simulates a terminal for users to perform simple commands such as getting files from a server and creating a history of all commands performed during a terminal session.
